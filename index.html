📂 Load Case Data
⚡ Quick Fetch — Enter Receipt Number
1Enter your receipt number above (starts with IOE followed by digits).
2Make sure you are already signed in to myaccount.uscis.gov in another tab of this same browser session.
3Click Open USCIS API — the raw JSON loads in a new tab. If the Case Status checkbox is ticked, a second tab opens with the case-status JSON (allow pop-ups for this site if your browser blocks it).

5Paste (Ctrl+V) into the text area above and click Analyze Case.

Note: due to browser security — Cross-Origin Resource Sharing (CORS) — direct auto-loading from an external domain is not possible. This step-by-step method is the fastest available path.

📖 How to manually access your USCIS case JSON data

To retrieve your case data in JSON format from the USCIS ELIS API:

1Sign in to your USCIS account at https://myaccount.uscis.gov/sign-in
2Open a new browser tab and navigate to (replace the receipt number with your own):

https://my.uscis.gov/account/case-service/api/cases/IOE09XXXXXXXX

3The browser displays the raw JSON response. Select all (Ctrl+A), copy (Ctrl+C), and paste (Ctrl+V) it into the text area above, then click Analyze Case.

You must be signed in to your USCIS account in the same browser session for the API link to work. The JSON is fetched directly from USCIS servers to your browser — this tool does not store or transmit any data.

🏛 USCIS service center codes — what your receipt prefix means

The three-letter prefix of a USCIS receipt number identifies where the case is processed. Some prefixes are older names for the same service center and still appear on existing case numbers.

Code Service center Notes
Current codes
VSCVermont Service CenterCurrent code USCIS uses for the Vermont Service Center.
CSCCalifornia Service CenterCurrent code for the California Service Center.
NSCNebraska Service CenterCurrent code for the Nebraska Service Center.
TSCTexas Service CenterCurrent code for the Texas Service Center.
PSCPotomac Service CenterCurrent code for the Potomac Service Center.
NBCNational Benefits CenterTechnically not a Service Center, but this code appears in receipt numbers for cases processed at the NBC.
IOEIntegrated Operating EnvironmentNot a physical service center. Used for electronic filings via the USCIS ELIS e-filing system, or when a paper application was scanned in at a service center — part of USCIS's move away from paper-based processing.
Previous codes — still found on older case numbers
ESCEastern Service CenterPrevious name/code for the Vermont Service Center (VSC).
EACEastern Adjudication CenterPrevious code for the Vermont Service Center (VSC).
WSCWestern Service CenterPrevious name/code for the California Service Center (CSC).
WACWestern Adjudication CenterPrevious code for the California Service Center (CSC).
LINLincoln Service CenterPrevious name/code for the Nebraska Service Center (NSC).
SSCSouthern Service CenterPrevious name/code for the Texas Service Center (TSC).
SRCSouthern Regional CenterPrevious code for the Texas Service Center (TSC).
YSCPotomac Service Center (previous code)Previous code for the Potomac Service Center (PSC); still found on case numbers processed there.
MSCNational Benefits Center (alternative code)Alternative/older code — cases starting with MSC are processed at the National Benefits Center (NBC).
